$D T = C T = \frac { 1 } { 2 } C D = 2 , \ D E = 2 D F$ $\frac { D E } { D F } { = } \frac { A D } { D T } { = } 2$ ，  
∵ $\angle E D F = \angle A D C = 9 0 ^ { \circ }$ ，$\angle E D A = \angle F D T$ ，

∵ $A T = \sqrt { A D ^ { 2 } + D T ^ { 2 } } = 2 \sqrt { 5 }$ ，
