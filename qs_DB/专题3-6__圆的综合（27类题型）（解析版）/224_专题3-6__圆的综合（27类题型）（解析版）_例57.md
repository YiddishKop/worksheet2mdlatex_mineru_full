$C E / / G M$ ，$\therefore \triangle G M H { \sim } \triangle C E H$ ，$\therefore { \frac { G H } { C H } } = { \frac { G M } { C E } } = { \frac { M H } { H E } } ,$ ∵ $C H = 3 G H$ ，$\therefore { \frac { 1 } { 3 } } = { \frac { G M } { 3 } } = { \frac { M H } { H E } } ,$ $G M = 1$ ，
