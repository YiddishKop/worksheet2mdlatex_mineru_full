(1)求证： $C F$ 是 $\odot O$ 切线；

(2)若 $A F = 1 0$ ， $\sin { F } = \frac { 2 } { 3 }$ ，求 $C D$ 的长． 【答案】(1)证明见解析 (2) $\frac { 8 { \sqrt { 5 } } } { 3 }$
