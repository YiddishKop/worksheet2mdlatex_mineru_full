（2）由圆周角定理得 $\angle A C B = 9 0 ^ { \circ }$ ， $\angle D A C = \angle P B C$ ，再证 $\angle B A C = \angle P B C$ ，然后证 $\triangle A C B \sim \triangle B C P$ ，得 ${ \frac { A C } { B C } } { = } { \frac { B C } { P C } }$ 即可得出结论；
