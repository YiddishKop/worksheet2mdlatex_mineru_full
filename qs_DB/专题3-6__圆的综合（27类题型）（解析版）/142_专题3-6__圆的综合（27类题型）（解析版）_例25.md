$A C = A B \cdot \sin B = 6 \times { \frac { 2 } { 3 } } = 4$ ，  
∴ $C E = 4$ ，  
$C F = C E \cdot \sin E = 4 \times { \frac { 2 } { 3 } } = { \frac { 8 } { 3 } }$   
由勾股定理得 $E F = { \sqrt { C E ^ { 2 } - C F ^ { 2 } } } = { \frac { 4 { \sqrt { 5 } } } { 3 } }$   
$A D = A E = 2 E F = \frac { 8 \sqrt { 5 } } { 3 }$
