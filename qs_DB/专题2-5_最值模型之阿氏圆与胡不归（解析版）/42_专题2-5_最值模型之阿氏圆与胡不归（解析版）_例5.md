【解答】（1）证明：四边形 $A B C D$ 是矩形  
$\therefore O D = O B = O C = O A$ ，  
 $\Delta E D C$ 和 $\Delta O D C$ 关于 $C D$ 对称，  
$\therefore D E = D O C E = C O$ ，  
$\therefore D E = E C = C O = O D$ ，
