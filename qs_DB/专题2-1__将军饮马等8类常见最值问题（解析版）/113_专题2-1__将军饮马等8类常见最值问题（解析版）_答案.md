【答案】 $\frac { 1 2 0 } { 1 7 } { \leq } C Q < 1 2$

【解答】由解析提示可知： $\frac { 5 } { 1 3 } \left( 1 2 - x \right) \leq x$ ，解得： $x \geq \frac { 6 0 } { 1 7 }$ ,所以 $\frac { 1 2 0 } { 1 7 } { \leq } C Q < 1 2$
