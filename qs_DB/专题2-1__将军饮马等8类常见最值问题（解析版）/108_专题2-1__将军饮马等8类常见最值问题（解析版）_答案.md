【答案】 $\frac { 2 5 } { 3 }$

延长 AB， $\frac { 5 } { 2 }$ ，取 CG 中点 M， $\mathrm { B F } \leqslant \mathrm { B M } = 5 \Rightarrow \mathrm { m } + \mathrm { n } \leqslant \frac { 2 5 } { 3 }$
