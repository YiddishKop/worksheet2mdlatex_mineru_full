设 $O F = x$ ，则 $A O = O D = x + 0 . 7$ ，  
又 $\cdot _ { \triangle A O E } \sim _ { \triangle C F E }$ ，${ \frac { A O } { O E } } { = } { \frac { C F } { E F } }$ , 即 $\frac { 0 . 7 + x } { 2 . 8 + x } = \frac { 2 . 1 } { 2 . 8 }$   
解得： $x = 5 . 6$ ，
