【详解】（1） $\cdot$ ∵抛物线 $y = a x ^ { 2 } + b x + c$ 与 $x$ 轴相交于点 $A ( - 1 , 0 )$ ，

∴抛物线的解析式为 $y { = } x ^ { 2 } - 2 x - 3$   
∵ $y = x ^ { 2 } - 2 x - 3 = ( x - 1 ) ^ { 2 } - 4$ ，  
∴点 $P$ 的坐标为 $( 1 , - 4 )$   
（2）由（1）知 $a - b + c = 0$ ，又 $3 b = 2 c$ ，  
$b = - 2 a , c = - 3 a \qquad ( a > 0 )$

∵ $y = a x ^ { 2 } - 2 a x - 3 a = a \left( x - 1 \right) ^ { 2 } - 4 a$ ，  
∴顶点 $\cdot$ 的坐标为 $( 1 , - 4 a )$
